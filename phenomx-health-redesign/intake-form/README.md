# PhenomX Health - Intake Form

A comprehensive health intake form designed for women's health assessment.

## Features
- Multi-step questionnaire
- Responsive design
- Progress tracking
- Data validation

## Development

\`\`\`bash
npm install
npm run dev
\`\`\`

## Build

\`\`\`bash
npm run build
\`\`\`

## Preview Production Build
